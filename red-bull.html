Ring toss

Inspired by motoraces, the package can be transform into the game of Ring Toss by tearing of the tires and using the can as a poll.

Frogger

This game uses an old chinese technic of origami to transform a part of the packaging into the jamping frog. The palyer can use the rest of the package as a score ring and try to send the frong into the ring.

Air hockey

Air Hockey is one of my absolute favorite table games and so is this packaging design. The players can transform the box into the playfield and use pop tops as a pack. Using the force of their breath, rivals can try to score a point into each other's gate.
